
## 1.斐波那契数列(特点：后一项等于前两项的和 1 1 2 3 5 8...)
```javascript
function fibonacci(n) {
  let n1 = 1, n2 = 1, sum;
  if (n <= 2) {
    return n1
  }
  for (let i = 2; i < n; i++) {
    sum = n1 + n2
    n1 = n2
    n2 = sum
  }
  return sum
}

fibonacci(30) //832040
```
## 2. 实现阶乘（5！= 5*4*3*2*1）
```javascript
function factorial(n) {
    let i = 1;
    let sum = 1;
    for (; i <= n; i++) {
        sum = sum * i;
    }
    return sum;
}
```
